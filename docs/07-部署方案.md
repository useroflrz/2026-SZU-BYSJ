# 部署方案（前端为主）

当前版本主要是前端（Vue + Cesium）能力验证与演示，可按静态站点方式部署。

## 1. 环境要求

- **Node.js**：用于构建（建议 16+）
- **静态服务器**：Nginx / IIS / 任意静态托管均可
- **网络**：若使用 Cesium Ion 世界地形/世界影像，需要可访问 Cesium 相关服务

## 2. 构建与部署

### 2.1 本地构建

在 `frontend/` 目录：

```bash
npm install
npm run build
```

构建产物通常在 `frontend/dist/`（以项目实际脚手架配置为准）。

### 2.2 静态部署（Nginx 示例）

- 将 `dist/` 目录内容部署到站点根目录
- 单页应用需要配置 history fallback（若使用了前端路由）

## 3. 配置项

### 3.1 Cesium Ion Token（可选）

可通过环境变量提供：

- `VITE_CESIUM_ION_TOKEN`

未配置时，应用会回退到椭球地形与无需 token 的底图方案（仍可进行北斗格网渲染与拾取验证）。
