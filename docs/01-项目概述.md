# 项目概述

## 1. 项目背景

本项目旨在将现有的QGIS Python工具（空域格网生成和可视域分析）迁移到Web平台，基于Vue + Cesium技术栈，实现一个可在浏览器中运行的3D GIS数据可视化与分析工具。

## 2. 项目目标

- **Web化部署**：将桌面端工具迁移到Web平台，支持跨平台访问
- **实时交互**：提供流畅的3D地图交互体验
- **自动化流程**：实现从区域选择到分析报告的完整自动化流程
- **高性能渲染**：支持大规模数据（百万级点）的实时渲染
- **易用性**：提供直观的用户界面，降低使用门槛

## 3. 核心功能

### 3.1 区域选择
- 支持选择任意城市/地区
- 支持手动绘制分析区域
- 支持导入区域边界（GeoJSON、KML等格式）
- 自动获取区域DSM数据

### 3.2 站点划设
- 在地图上交互式添加站点（发射源）
- 支持批量导入站点数据
- 站点属性配置（高度、频段等）
- 站点可视化标记

### 3.3 空域格网生成
- 自动切分空域网格
- 支持自定义格网尺寸（长×宽×高）
- 支持北斗网格剖分标准
- 自动叠加DSM高程数据
- 生成3D格网中心点

### 3.4 LOS可视域分析
- 视线（Line of Sight）分析
- 支持默认参数快速分析
- 支持复杂参数配置（信号衰减、地球曲率等）
- 批量处理大规模接收点
- 实时分析进度反馈

### 3.5 可视化渲染
- 3D地形渲染（基于DSM）
- 格网点云渲染
- LOS分析结果可视化（颜色编码）
- 站点和接收点标记
- 支持图层控制（显示/隐藏、透明度）

### 3.6 报告生成
- 自动生成分析报告
- 统计信息汇总（可视比例、覆盖率等）
- 支持导出PDF、Excel格式
- 可视化图表（统计图表、热力图等）

## 4. 技术特点

- **前端**：Vue 3 + Cesium（3D地球引擎）
- **后端**：Python FastAPI（处理GIS计算）
- **数据格式**：GeoJSON、GeoTIFF、GeoPackage
- **坐标系统**：支持投影坐标系（如EPSG:4547）
- **性能优化**：Web Worker并行计算、数据分块加载

## 5. 用户场景

1. **规划人员**：选择城市区域，划设基站站点，生成空域格网
2. **分析人员**：配置分析参数，执行LOS分析，查看可视化结果
3. **决策人员**：查看分析报告，评估覆盖效果，导出数据

## 6. 项目优势

- **无需安装**：基于Web，打开浏览器即可使用
- **实时协作**：支持多用户同时访问
- **数据共享**：分析结果可导出，便于分享
- **扩展性强**：模块化设计，易于扩展新功能
