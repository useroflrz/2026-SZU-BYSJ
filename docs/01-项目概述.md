# 项目概述

## 1. 项目背景

本项目基于 Vue 3 + Cesium 构建浏览器端 3D GIS 可视化工具，核心目标是完成 **北斗格网（3D 空域格网）在中小规模与大规模场景下的高性能渲染与交互拾取**，为后续分析模块（如可视域/覆盖计算）提供可扩展的前端渲染基础。

## 2. 项目目标

- **Web化部署**：将桌面端工具迁移到Web平台，支持跨平台访问
- **实时交互**：提供流畅的3D地图交互体验
- **高性能渲染**：支持中小规模与大规模北斗格网的实时渲染与拾取
- **易用性**：提供直观的用户界面，降低使用门槛

## 3. 核心功能

### 3.1 区域选择
- 支持城市/地区选择（内置示例）
- 支持导入 SHP（推荐 zip：包含 `.shp/.dbf/.shx/.prj`）并解析范围（bbox）
- 将选定区域写入前端状态，供格网参数联动与渲染使用

### 3.2 站点划设
- 支持站点生成与批量导入（用于后续分析/展示）
- 站点可视化标记与列表管理

### 3.3 空域格网生成
- 支持设置格网参数：\(dx, dy, dz, zMin, zMax\)
- 支持两种渲染策略（自动按规模切换）：
  - **中小规模**：Cesium `GeometryInstance + Primitive`（可直接 `scene.pick` 拾取）
  - **大规模**：自定义 `BeiDouGridPrimitive`，通过 **GPU 实例化绘制**（支持百万级）并提供 GPU 拾取

### 3.5 可视化渲染
- 3D 地图与基础底图展示
- 北斗格网渲染（面颜色/透明度、边线颜色/透明度）
- 格网单元拾取与高亮显示（中小规模走 Cesium pick，大规模走 GPU pick）

## 4. 技术特点

- **前端**：Vue 3 + Cesium（项目约定：`cesium@1.99.0`）
- **数据格式**：区域边界（SHP/zip），站点（CSV/GeoJSON）
- **坐标系统**：前端渲染与区域范围以经纬度（WGS84 / EPSG:4326）为主
- **性能关键**：大规模格网使用自定义 Primitive + GPU 实例化（单几何、多实例矩阵）

## 5. 用户场景

1. **业务人员/研究人员**：选择区域（城市或导入 SHP），配置格网参数，生成并渲染 3D 北斗格网
2. **开发/测试人员**：验证不同规模下的渲染性能与拾取正确性（中小规模 vs 大规模）

## 6. 项目优势

- **无需安装**：基于Web，打开浏览器即可使用
- **性能可控**：按规模自动切换渲染策略，兼顾正确性与性能
- **扩展性强**：渲染与交互链路清晰，便于接入更多分析/可视化能力
