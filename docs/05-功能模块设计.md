# 功能模块设计文档

## 1. 区域选择模块

### 1.1 功能描述

支持用户选择分析区域，包括城市选择、手动绘制、文件导入等方式。

### 1.2 实现方案

#### 前端实现
- **城市选择器**：下拉选择预设城市，自动定位到该城市
- **地图绘制工具**：使用Cesium的绘制工具，支持矩形、多边形绘制
- **文件上传**：支持GeoJSON、KML格式的区域边界文件

#### 后端实现
- **区域数据管理**：维护城市区域数据库（名称、边界、CRS）
- **文件解析**：使用GDAL/OGR解析上传的区域文件
- **坐标转换**：统一转换为项目坐标系（如EPSG:4547）

### 1.3 数据流程

```
用户选择/绘制区域 
→ 前端发送区域边界 
→ 后端验证并存储 
→ 返回区域ID和DSM信息 
→ 前端加载DSM地形
```

## 3. 空域格网生成模块

### 3.1 功能描述

根据配置参数自动生成空域格网中心点，叠加DSM高程数据。

### 3.2 核心算法

#### 3.2.1 规则网格生成

```python
def generate_regular_grid(bounds, dx, dy, dz, z_min, z_max):
    """
    生成规则网格
    
    参数:
        bounds: 区域边界 {minX, minY, maxX, maxY}
        dx, dy: 水平格网尺寸（米）
        dz: 垂直层高间隔（米）
        z_min, z_max: 高度范围（相对高度，米）
    
    返回:
        格网点列表
    """
    z_levels = np.arange(z_min, z_max + dz/2, dz)
    x_coords = np.arange(bounds['minX'] + dx/2, bounds['maxX'], dx)
    y_coords = np.arange(bounds['minY'] + dy/2, bounds['maxY'], dy)
    
    points = []
    for layer_idx, z in enumerate(z_levels):
        for x_idx, x in enumerate(x_coords):
            for y_idx, y in enumerate(y_coords):
                # 采样DSM高程
                dsm_z = sample_dsm(x, y)
                if dsm_z is not None:
                    z_abs = dsm_z + z
                    points.append({
                        'x': x, 'y': y, 'z': z_abs,
                        'dsm_z': dsm_z,
                        'z_layer': z,
                        'grid_x_id': x_idx,
                        'grid_y_id': y_idx,
                        'layer_id': layer_idx
                    })
    return points
```

### 3.3 DSM高程采样

```python
def sample_dsm(x, y, dsm_data):
    """
    从DSM数据中采样高程值
    
    参数:
        x, y: 投影坐标
        dsm_data: DSM数据字典
    
    返回:
        高程值（米）或None（NoData）
    """
    extent = dsm_data['extent']
    array = dsm_data['array']
    cell_size_x = dsm_data['cell_size_x']
    cell_size_y = dsm_data['cell_size_y']
    
    # 计算像素坐标
    col = int((x - extent.xMinimum()) / cell_size_x)
    row = int((extent.yMaximum() - y) / cell_size_y)
    
    # 边界检查
    if 0 <= row < array.shape[0] and 0 <= col < array.shape[1]:
        value = array[row, col]
        if not np.isnan(value):
            return float(value)
    
    return None
```

### 3.4 性能优化

- **并行处理**：使用多进程处理不同高度层
- **分块处理**：大区域分块处理，避免内存溢出
- **缓存机制**：缓存DSM数据，避免重复读取

## 4. LOS可视域分析模块

### 4.1 功能描述

计算接收点与发射源之间的视线（Line of Sight），判断是否可视。

### 4.2 核心算法

#### 4.2.1 视线检查算法

```python
def check_line_of_sight(dsm_data, tx_x, tx_y, tx_height,
                        rx_x, rx_y, rx_height, distance,
                        consider_curvature=True):
    """
    检查两点间是否存在视线
    
    参数:
        dsm_data: DSM数据
        tx_x, tx_y, tx_height: 发射端坐标和高度
        rx_x, rx_y, rx_height: 接收端坐标和高度
        distance: 两点间距离（米）
        consider_curvature: 是否考虑地球曲率
    
    返回:
        bool: True表示可视，False表示不可视
    """
    if distance < 10:
        return True
    
    # 使用Bresenham算法遍历路径上的像素
    x0 = world_to_pixel(tx_x, dsm_data)
    y0 = world_to_pixel(tx_y, dsm_data)
    x1 = world_to_pixel(rx_x, dsm_data)
    y1 = world_to_pixel(rx_y, dsm_data)
    
    height_diff = rx_height - tx_height
    
    for x, y in bresenham_line(x0, y0, x1, y1):
        if x == x0 and y == y0:
            continue  # 跳过起点
        
        terrain_h = dsm_data['array'][y, x]
        if np.isnan(terrain_h):
            continue
        
        # 计算路径上该点的视线高度
        ratio = calculate_ratio(x, y, x0, y0, x1, y1)
        line_height = tx_height + height_diff * ratio
        
        # 考虑地球曲率
        if consider_curvature:
            current_dist = distance * ratio
            curvature = calculate_earth_curvature(current_dist)
            line_height -= curvature
        
        # 判断是否被遮挡
        if terrain_h > line_height + 3.0:  # 3米安全余量
            return False
    
    return True
```

#### 4.2.2 地球曲率计算

```python
def calculate_earth_curvature(distance_m):
    """
    计算地球曲率影响
    
    参数:
        distance_m: 距离（米）
    
    返回:
        曲率高度（米）
    """
    earth_radius = 6371000  # 地球半径（米）
    effective_radius = earth_radius * 4 / 3  # 考虑大气折射
    return (distance_m ** 2) / (2 * effective_radius)
```

#### 4.2.3 路径损耗计算

```python
def calculate_path_loss(distance_km, frequency_ghz):
    """
    计算自由空间路径损耗
    
    参数:
        distance_km: 距离（千米）
        frequency_ghz: 频段（GHz）
    
    返回:
        路径损耗（dB）
    """
    return 20 * math.log10(distance_km) + \
           20 * math.log10(frequency_ghz) + 92.45
```

### 4.3 批量处理优化

```python
def batch_los_analysis(receivers, transmitters, dsm_data, config):
    """
    批量LOS分析
    
    优化策略:
    1. 按距离排序发射源，优先检查最近的
    2. 找到第一个可视即停止
    3. 使用路径损耗预过滤
    4. 并行处理多个接收点
    """
    results = []
    
    for receiver in receivers:
        rx_coords = np.array([receiver.x, receiver.y])
        
        # 计算到所有发射源的距离
        tx_coords = np.array([(tx.x, tx.y) for tx in transmitters])
        distances = np.sqrt(np.sum((tx_coords - rx_coords) ** 2, axis=1))
        
        # 按距离排序
        sorted_indices = np.argsort(distances)
        
        found_visible = False
        for idx in sorted_indices:
            distance = distances[idx]
            
            # 距离过滤
            if distance > config['max_distance']:
                break
            
            # 路径损耗过滤
            path_loss = calculate_path_loss(
                distance / 1000.0,
                config['frequency']
            )
            if path_loss > config['path_loss_threshold']:
                continue
            
            # 视线检查
            tx = transmitters[idx]
            is_los = check_line_of_sight(
                dsm_data,
                tx.x, tx.y, tx.height,
                receiver.x, receiver.y, receiver.height,
                distance,
                config['consider_curvature']
            )
            
            if is_los:
                results.append({
                    'receiver_id': receiver.id,
                    'visible': True,
                    'distance': distance,
                    'path_loss': path_loss,
                    'closest_tx': tx.id
                })
                found_visible = True
                break
        
        if not found_visible:
            results.append({
                'receiver_id': receiver.id,
                'visible': False
            })
    
    return results
```

### 5.2 渲染策略

#### 5.2.1 点云渲染

```javascript
// 使用PointPrimitiveCollection批量渲染
function renderGridPoints(points) {
  const collection = viewer.scene.primitives.add(
    new Cesium.PointPrimitiveCollection()
  )
  
  points.forEach(point => {
    const color = point.visible 
      ? Cesium.Color.GREEN 
      : Cesium.Color.RED
    
    collection.add({
      position: Cesium.Cartesian3.fromDegrees(
        point.lon, point.lat, point.height
      ),
      color: color,
      pixelSize: 3,
      outlineColor: Cesium.Color.WHITE,
      outlineWidth: 1
    })
  })
  
  return collection
}
```
