# 图8 研究区域与站点分布（截图与合成模板）

## 主图采集（Cesium）

- 图层开启：影像 + 地形晕渲 + 实验边界 + 5个站点
- 边界样式：红色多边形描边
- 站点区分：按高程使用不同颜色或图标尺寸
- 视角建议：中等俯视，完整覆盖南山区实验边界

主图文件名建议：`fig8-main-cesium.png`

## 位置小图采集（QGIS）

- 深圳市行政区底图
- 高亮南山区或实验区域位置
- 输出透明底或白底小图

小图文件名建议：`fig8-inset-shenzhen.png`

## 合成步骤（Photoshop）

1. 画布建议 `2200x1500`。
2. 放入主图，并在右下角嵌入位置小图。
3. 添加图例：边界、站点类型/高程等级。
4. 添加文本标注：
   - `高程范围 0-430m`
   - `面积约 180km²`
   - `格网分辨率 100m`
5. 导出 `PNG 300dpi`。
