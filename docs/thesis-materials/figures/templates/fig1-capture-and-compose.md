# 图1 边界对齐效果对比（截图与合成模板）

## 截图前固定参数

- 分辨率：`3840x2160`（与论文偏移验证一致）
- 相机：同一经纬度、同一俯仰角、同一缩放级别
- 边界线宽：固定（建议 2px）
- 底图与地形源：两组对比必须保持一致

## 需要截取的两张图

1. `fig1-left-system.png`：本系统同源数据链路（期望 0 像素偏移）
2. `fig1-right-traditional.png`：传统分离链路（可见 0.5~2 像素偏移）

## 后期合成步骤（Photoshop）

1. 新建画布 `2400x1200`，左右分栏各占 50%。
2. 左图标题：`本系统（同源）`；右图标题：`传统方案（分离）`。
3. 在两图中选同一局部边界区域，添加放大框与连线。
4. 放大框中叠加像素标尺（可用标尺工具截图）。
5. 左图标注：`0像素偏移`；右图标注：`最大2像素偏移`。
6. 导出：
   - 可编辑：`fig1-boundary-compare.psd`
   - 论文图：`fig1-boundary-compare.png`（300dpi）

## 通过标准

- 左右图地理位置完全一致
- 放大框内边界对齐差异可肉眼区分
- 图中文字与论文口径完全一致
