# 论文12图统一图题与图注（定稿模板）

图1 边界对齐效果对比图  
左为本系统同源链路，矢量边界与计算掩膜重合；右为传统分离方案，局部出现像素级偏移与锯齿。

图2 规则三维格网与格柱结构示意图  
展示规则体素格网在 `i,j,k` 索引下的组织方式，以及同 `(i,j)` 方向上的格柱聚合与掩膜剔除逻辑。

图3 自适应视线采样策略原理图  
示意发射点与接收点连线、地形剖面插值和非均匀采样分布关系，并给出可见性判定规则。

图4 双模式渲染切换机制流程图  
依据活跃格元数阈值 `1.2×10^5` 在小规模与大规模渲染路径间切换，实现统一拾取输出。

图5 系统总体架构图  
给出应用层、服务层、数据层的职责划分，以及从用户输入到结果渲染的主数据流。

图6 异步计算任务状态时序图  
展示任务提交、`task_id` 返回、500ms轮询状态、结果回传与前端重建渲染全过程。

图7 边界-地形联动数据同源机制图  
说明掩膜判定与地形采样并行生成共享数据结构，并被渲染与LOS计算同时复用。

图8 研究区域与站点分布图  
以深圳市南山区为实验区域，标注边界范围、站点分布及高程区间等实验背景信息。

图9 可视域分析结果三维效果图  
主图显示可见与不可见格网分布，小图补充站点视线锥与分层统计结果。

图10 渲染性能对比折线图  
对比不同活跃格元规模下两种渲染策略的帧率变化，并标注阈值与低帧率区间。

图11 端到端流程耗时堆叠柱状图  
比较本系统与传统方案在数据准备、采样、计算和可视化各阶段耗时构成及总耗时差异。

图12 计算准确率对比图（混淆矩阵）  
以 ArcGIS 结果为参照展示四类判定结果分布，突出总体一致率 `94.2%` 与误差来源。
