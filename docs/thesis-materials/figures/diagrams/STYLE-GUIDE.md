# 图2~图7示意图统一风格模板

## 版式

- 画布比例：`16:9`（推荐 1920x1080）
- 主流程方向：从左到右
- 结构层次：输入在左，处理在中，输出在右

## 字体与字号

- 中文：思源黑体 / 黑体
- 英文数字：Arial
- 标题：28
- 一级节点：20
- 二级注释：16

## 配色约定

- 输入节点：浅蓝
- 处理节点：浅黄
- 数据结构节点：浅紫
- 输出节点：浅绿
- 无效区或剔除区：灰色
- 关键阈值/公式：红色文字强调

## 标注约定

- 变量统一使用：`N`, `i,j,k`, `Δx,Δy,Δz`, `task_id`, `status`, `progress`
- 技术栈节点必须显示全称：`Vue3`, `Pinia`, `Cesium 1.99.0`, `FastAPI`, `Uvicorn`, `NumPy`
- 复杂度表达固定写法：`Draw Call O(N) -> O(1)`

## 导出建议

- 流程图优先导出矢量：`SVG` 或 `PDF`
- 入论文位图：`PNG 300dpi`
