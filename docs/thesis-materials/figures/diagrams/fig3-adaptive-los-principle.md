# 图3 自适应视线采样策略原理图（制作草图）

## 图面布局（二维剖面）

- 左侧主图：`T -> R` 视线与地形剖面
- 右侧小图：采样密度/步长随距离变化曲线

## 主图必须包含

1. 发射点 `T`（高处）与接收点 `R`（目标格元）。
2. 地形高程曲线（可标注“双线性插值地形”）。
3. 视线段（直线）。
4. 沿视线的采样点：近处较疏、远处较密。
5. 遮挡判定注释：`h_line(d) > h_terrain(d) => 可见`。

## 右侧小图（函数可视化）

- 横轴：距离 `d`
- 纵轴：采样点数或采样密度 `n(d)`
- 曲线趋势：随距离增加而上升
- 标注“对应公式(5)”

## Python 草图建议

1. 用 `numpy` 构造一条起伏地形曲线。
2. 用线性插值生成 LOS 高度线。
3. 人工构造一组非均匀采样位置（远处更密）。
4. 使用 `matplotlib` 两子图并排导出 `SVG`。
5. 在 Illustrator 统一中文标注与箭头样式。
