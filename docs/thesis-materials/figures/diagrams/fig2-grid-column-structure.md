# 图2 规则三维格网与格柱结构示意（制作草图）

## 图面布局（2.5D 透视）

1. 左侧放完整体素块（显示 `i,j,k` 索引方向）。
2. 中间高亮一个固定 `(i,j)` 的竖向列，标注为 `GridColumn(i,j)`。
3. 右侧给出掩膜剔除示意：灰色区域表示 `columnMask=false`。
4. 顶部放步长标注：`Δx`, `Δy`, `Δz`。
5. 底部放中心点公式简写：

`center(i,j,k) = (x0 + (i+0.5)Δx, y0 + (j+0.5)Δy, z0 + (k+0.5)Δz)`

## 元素清单

- 坐标轴箭头：`X(经向)`, `Y(纬向)`, `Z(高度)`
- 索引标签：`i`, `j`, `k`
- 图例：
  - 蓝色：有效格元
  - 灰色：掩膜剔除格元
  - 橙色描边：聚合后的格柱

## 推荐工具操作（PowerPoint 3D）

1. 插入立方体阵列（复制对齐形成 6x6x4 示例）。
2. 统一透明度到 35%，边框 1.2pt。
3. 选中同一 `(i,j)` 的 4 个立方体，改边框色为橙色并加外框括号。
4. 将左下角 2x2 区域改为灰色，标注 `mask=false`。
5. 添加箭头和公式文本框，导出 `SVG` 与 `PNG`。
