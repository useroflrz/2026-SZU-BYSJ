## 中文摘要（建议结构，2～3 段）

**第一段（背景与问题）**  
空域管理与电磁环境分析常需要在三维空间中对目标区域进行离散化表达，并在地形约束下进行通视性（Line of Sight, LOS）判断。传统桌面 GIS 工具在部署与协同上成本较高；而在浏览器端基于 WebGL 的三维地球引擎渲染海量三维格网时，易出现帧率下降、交互拾取困难、长时间分析阻塞界面等问题。

**第二段（方法与系统）**  
本文设计并实现了一套基于 Vue 3 与 Cesium（1.99.0）的 Web 三维 GIS 系统，联合 FastAPI 后端完成格网场景下的可视域分析。前端支持城市或矢量（Shapefile zip）划定范围，配置三维格网参数并完成站点管理；针对格网规模采用**双路径渲染策略**——总格元不超过 `120000` 时使用 `GeometryInstance` 与原生拾取，超过阈值时切换**自定义 Primitive 与 GPU 实例化绘制**，并通过**离屏实例 ID 缓冲**实现拾取与高亮。可视域计算在后端以**异步作业**方式执行，前端按约 `500ms` 周期轮询进度；计算侧采用地形高程栅格双线性插值、LOS 路径自适应采样及距离内候选站点预筛等策略，避免长时间占用请求线程并便于前端展示进度。

**第三段（结果与意义，需你补实测）**  
实验表明，该系统能够在典型配置下完成（请填写：例如某城市范围、某 dx/dy/dz 下）格网可视化与可视域统计；在性能实验中可按“预热 + 采样”的统一流程得到可复现 FPS（请填写：关键帧率与规模上限）；在分析实验中可按作业创建、状态轮询和结果拉取记录端到端耗时（请填写：典型耗时与结果规模）。系统为后续扩展 DSM 服务化、报告导出与多算法对比预留了接口与模块边界。
