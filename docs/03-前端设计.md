## 2. 核心组件设计

### 2.1 MapViewer 组件

**功能**：Cesium地图容器，负责3D场景渲染

**Props**:
```javascript
{
  center: { lon: Number, lat: Number },  // 初始中心点
  zoom: Number,                          // 初始缩放级别
  terrainProvider: Object,               // 地形提供者
  layers: Array                         // 图层列表
}
```

**主要方法**:
- `loadDSM(url)` - 加载DSM地形
- `addPointLayer(data)` - 添加点图层
- `addStation(position)` - 添加站点标记
- `flyTo(extent)` - 飞行到指定区域
- `clearLayers()` - 清除所有图层

### 2.2 RegionSelector 组件

**功能**：区域选择组件

**功能点**:
- 城市/地区下拉选择
- 地图绘制矩形/多边形区域
- 导入区域边界文件（GeoJSON/KML）
- 显示区域边界和面积

**数据格式**:
```javascript
{
  name: String,
  bounds: { minX, minY, maxX, maxY },
  geometry: GeoJSON,
  crs: String
}
```

### 2.3 StationManager 组件

**功能**：站点管理组件

**功能点**:
- 地图上点击添加站点
- 站点列表管理（增删改查）
- 站点属性编辑（名称、高度、频段）
- 批量导入站点（CSV/GeoJSON）
- 站点可视化标记

**数据模型**:
```javascript
{
  id: String,
  name: String,
  position: { lon, lat, height },
  frequency: Number,      // GHz
  height: Number,          // 绝对海拔（米）
  type: String            // 站点类型
}
```

### 2.4 GridConfig 组件

**功能**：格网配置组件

**配置项**:
- 格网尺寸：宽度(DX)、高度(DY)、层高间隔(DZ)
- 高度范围：最低高度(Z_MIN)、最高高度(Z_MAX)
- 网格类型：规则网格 / 北斗网格
- 自动使用DSM边界开关

**参数验证**:
- DX、DY、DZ > 0.1米
- Z_MIN < Z_MAX
- 范围需大于格网尺寸

### 2.5 AnalysisPanel 组件

**功能**：LOS分析配置面板

**配置项**:
- **基础参数**：
  - 发射端高度字段选择
  - 接收端高度字段选择
  - 频段设置（GHz）
  
- **高级参数**：
  - 是否考虑地球曲率
  - 路径损耗阈值（dB）
  - 批处理大小
  - 最大分析距离

**分析模式**:
- **快速模式**：使用默认参数
- **高级模式**：自定义所有参数

### 2.6 ResultVisualizer 组件

**功能**：分析结果可视化组件

**可视化方式**:
- **点云渲染**：根据可视性着色（绿色=可视，红色=不可视）
- **热力图**：基于可视比例生成热力图
- **统计图表**：可视比例、距离分布等
- **图层控制**：显示/隐藏、透明度调整

**性能优化**:
- 使用Cesium的`PointPrimitiveCollection`批量渲染
- 根据相机距离动态调整点大小
- 支持按层高筛选显示

## 3. 状态管理设计

### 3.1 Map Store (map.js)

```javascript
{
  state: {
    viewer: null,              // Cesium Viewer实例
    center: { lon, lat },
    zoom: Number,
    layers: [],                // 图层列表
    selectedRegion: null,      // 选中的区域
    dsmLoaded: false          // DSM是否已加载
  },
  actions: {
    initViewer(),
    loadDSM(),
    addLayer(),
    removeLayer(),
    setRegion()
  }
}
```

### 3.2 Analysis Store (analysis.js)

```javascript
{
  state: {
    stations: [],              // 站点列表
    gridPoints: [],            // 格网点数据
    analysisResult: null,      // 分析结果
    analysisProgress: 0,       // 分析进度
    isAnalyzing: false         // 是否正在分析
  },
  actions: {
    addStation(),
    removeStation(),
    generateGrid(),
    runAnalysis(),
    clearResults()
  }
}
```

## 4. Cesium集成方案

### 4.1 初始化配置

```javascript
import * as Cesium from 'cesium'

// 设置Cesium访问令牌（需要注册获取）
Cesium.Ion.defaultAccessToken = 'your-token'

// 创建Viewer
const viewer = new Cesium.Viewer('cesiumContainer', {
  terrainProvider: Cesium.createWorldTerrain(),
  imageryProvider: new Cesium.OpenStreetMapImageryProvider({
    url: 'https://a.tile.openstreetmap.org/'
  }),
  baseLayerPicker: false,
  animation: false,
  timeline: false,
  vrButton: false
})
```

### 4.2 DSM地形加载

```javascript
// 方式1：使用TerrainProvider加载GeoTIFF
const terrainProvider = new Cesium.CesiumTerrainProvider({
  url: '/api/terrain/{z}/{x}/{y}'
})

// 方式2：使用3D Tiles加载处理后的DSM
const tileset = viewer.scene.primitives.add(
  new Cesium.Cesium3DTileset({
    url: '/api/tileset/dsm'
  })
)
```

### 4.3 点云渲染

```javascript
// 使用PointPrimitiveCollection批量渲染
const pointCollection = viewer.scene.primitives.add(
  new Cesium.PointPrimitiveCollection()
)

// 添加点
gridPoints.forEach(point => {
  pointCollection.add({
    position: Cesium.Cartesian3.fromDegrees(
      point.lon, point.lat, point.height
    ),
    color: point.visible ? Cesium.Color.GREEN : Cesium.Color.RED,
    pixelSize: 3
  })
})
```

### 4.4 交互功能

```javascript
// 绘制矩形区域
const handler = new Cesium.ScreenSpaceEventHandler(viewer.scene.canvas)
handler.setInputAction((click) => {
  // 绘制逻辑
}, Cesium.ScreenSpaceEventType.LEFT_CLICK)

// 添加站点标记
const entity = viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(lon, lat, height),
  billboard: {
    image: '/assets/station-icon.png',
    scale: 1.0
  },
  label: {
    text: stationName,
    font: '14px sans-serif'
  }
})
```

## 5. 性能优化策略

### 5.1 数据分块加载

```javascript
// 将大规模点云分块加载
async function loadGridPointsChunked(points, chunkSize = 10000) {
  for (let i = 0; i < points.length; i += chunkSize) {
    const chunk = points.slice(i, i + chunkSize)
    await renderPoints(chunk)
    // 让UI有机会更新
    await new Promise(resolve => setTimeout(resolve, 10))
  }
}
```

### 5.2 LOD机制

```javascript
// 根据相机距离调整点大小和密度
viewer.camera.moveEnd.addEventListener(() => {
  const height = viewer.camera.positionCartographic.height
  const pixelSize = height > 10000 ? 2 : 5
  pointCollection.pixelSize = pixelSize
})
```

### 5.3 Web Worker计算

```javascript
// 在Worker中处理复杂计算
const worker = new Worker('/workers/grid-calculator.js')
worker.postMessage({ points, stations, dsm })
worker.onmessage = (e) => {
  const results = e.data
  updateVisualization(results)
}
```

## 6. UI/UX设计原则

- **直观性**：操作流程清晰，减少学习成本
- **反馈性**：所有操作都有明确的进度和结果反馈
- **响应性**：界面响应迅速，避免长时间等待
- **一致性**：保持设计风格和交互方式一致
- **可访问性**：支持键盘操作，提供无障碍支持
