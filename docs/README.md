## 注意事项

1. **Cesium 版本**：项目约定使用 `cesium@1.99.0`
2. **坐标系**：前端区域/渲染以经纬度（WGS84/EPSG:4326）为主；格网尺寸参数单位为米
3. **性能考虑**：大规模格网使用自定义 Primitive + GPU 实例化渲染；中小规模使用 GeometryInstance 以获得更直接的拾取能力
